## 📖 文档维护

### 文档原则
- **中文优先**：所有文档使用中文编写
- **结构清晰**：使用清晰的标题和层级
- **示例丰富**：提供实用的代码示例
- **保持更新**：代码变更时同步更新文档

### 文档更新流程
1. 识别需要更新的文档
2. 修改内容并测试示例
3. 更新相关文档的交叉引用
4. 提交 PR 时在描述中说明文档变更
