# AgentMesh 贡献者许可协议(CLA)

> 在向 AgentMesh 项目(以下简称"本项目")提交贡献前,请阅读并签署本协议。
>
> 签署流程:首次提交 PR 时,[CLA Assistant](https://cla-assistant.io/) 机器人会在 PR 中评论并提供签署链接,使用你的 GitHub 账号一键签署即可。

## 1. 定义

- **"你"** 或 **"贡献者"** 指代表自己或所代表实体签署本协议的法人或自然人。
- **"项目方"** 指 AgentMesh 项目的维护团队。
- **"贡献"** 指你以任何形式(代码、文档、设计、测试、issue 等)提交给本项目的原创作品。

## 2. 版权许可

你授予项目方一项**不可撤销的、永久的、全球范围的、免费的、可分许可的、非排他性的**许可,允许项目方:

- 复制、修改、合并、分发、公开展示、公开演示你的贡献;
- 将你的贡献整合到任何衍生作品中,包括但不限于开源或闭源软件;
- 通过任何媒介(包括但不限于 PyPI、GitHub、Docker Hub、商业软件包)再分发。

## 3. 专利许可

对于你的贡献中所涉及的、由你拥有或控制的任何专利权利要求,你授予项目方及本项目最终用户**不可撤销的、永久的、全球范围的、免费的**许可,允许制造、使用、销售、提供销售、进口和以其他方式转移你的贡献。

如果任何人就本项目或本项目所整合的贡献对你提起专利诉讼(包括反诉),则根据本协议授予该人的专利许可将于该诉讼提起之日起终止。

## 4. 商业使用授权

**项目方有权将你的贡献用于任何商业目的,包括但不限于:**

- 纳入商业版本或闭源衍生作品;
- 用于提供付费服务、企业版、SaaS;
- 与第三方达成商业合作,并将贡献纳入合作产品。

你理解并同意,签署本协议后,你**不会就项目方对贡献的商业使用提出任何额外的补偿要求**。

## 5. 原创性声明

你声明并保证:

- 你的贡献是你的原创作品,**或者** 你有权根据本协议的条款提交该贡献;
- 你的贡献不侵犯任何第三方的版权、专利、商标、商业秘密或其他知识产权;
- 如果你的贡献包含或基于第三方材料(如开源代码片段),你已在贡献中明确标注其出处和许可证;
- 如果你代表雇主提交贡献,你已获得雇主的明确授权。

## 6. 你保留的权利

签署本协议后,**你仍然保留对你的贡献的版权**。本协议为非排他性授权,你可以将贡献用于其他项目或自行许可给第三方。

## 7. 担保免责

你的贡献**按"原样"提供**,你不对其作出任何明示或暗示的担保,包括但不限于适销性、特定用途适用性或不侵权的担保。

## 8. 协议变更

本协议可能会更新。重大变更将通过 GitHub Discussions 或公开通告通知贡献者。继续向本项目提交贡献,即视为接受更新后的版本。

## 9. 适用法律

本协议受**中华人民共和国法律**管辖。因本协议引起的或与之相关的任何争议,应首先通过友好协商解决;协商不成的,提交至项目方所在地有管辖权的人民法院。

## 10. 完整协议

本协议构成你与项目方就贡献事项达成的完整协议,取代之前的所有口头或书面理解。
